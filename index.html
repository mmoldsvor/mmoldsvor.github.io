Disruptive Technologies Cloud HomeKit Implementation (2020)

Uses Homebridge API to set up Disruptive Sensors as Apple HomeKit accessories. Runs a Web Server on your Raspberry Pi that receives HTTP-requests from Disruptive Data Connectors. Gives users the ability to integrate Disruptive Sensors in their Smart Home.

Home Automation (2019)

A RESTful API created to control all devices using HTTP requests.The first HTTP client I created to control smart devices with Disruptive Technologies touch sensors. Created to be expandable for many types of devices.

Supports Disruptive Technologies touch sensors, Telldus Live! Smart Equipment and Phillips Hue Smart Bulbs.
